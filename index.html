Настройка Firebase

Для работы с облачным хранением данных необходимо настроить Firebase:

  1. Перейдите на Firebase Console
  2. Создайте новый проект
  3. В настройках проекта найдите конфигурацию и вставьте её ниже

Настройка провайдеров аутентификации

Для работы авторизации через Google и Яндекс необходимо настроить провайдеров:

  1. В Firebase Console перейдите в раздел "Authentication"
  2. На вкладке "Sign-in method" нажмите "Google" и включите его
  3. Укажите публичное название и email поддержки
  4. Сохраните настройки
  5. Для Яндекс: Выберите "Настроить домен" в настройках проекта Firebase

Тестовый аккаунт Firebase

Для быстрого тестирования можно использовать:

Email: test@example.com

Пароль: test123456

Или создайте свой аккаунт через форму регистрации
